ОСНОВНОЕ ПРИЛОЖЕНИЕ STB820

2008-04

* Добавлена настройка HTTP Proxy, которая задает прокси-сервер для браузера.
* Добавлено сообщение "формат видео не поддерживается" в Эфирное вещание
* NWSource больше не используется
* Добавлен SAP-приемник Платона
* Добавлено сообщение "нет данных" в VOD, отображающееся при длительном отсутствии данных из сети
* Добавлен легковесный RTSP-клиент вместо NWSource
* Добавлена функция роутера для второго сетевого адаптера.
* Добавлено автоматическое распознование типа вещания (RTP/UDP) в случае MPEG TS
* Исправлена ошибка определения типа вещания (UDP/RTP) в при работе с VOD
* Убран выбор папок в "сохраненном видео"

2008-03

* Исправлено падение приложения в случае возникновения ошибки при инициализации приема вещания
* Добавлены дополнительные настройки сканированитя DVB-каналов (доступны при наличии тюнера)
* Добавлена поддержка комманд пульта "Channel +/Channel -"
* Добавлена возможность текстового ввода с пульта (только английские символы)
* Исправлено зависание приложения при нажатии паузы после завершения видео (особенности vod-сервера)
* Добавлена возможность применения сетевых настроек без перезагрузки
* Оптимизировано отображение слайдера (экономия памяти)
* Изменено положение кнопок управления воспроизведением (чтобы были как на пульте)
* Обновлены иконки
* Использование програмного демуксера
* Использование медленного сканирования DVB-каналов

2008-02

* Улучшена функция ввода текста с пульта
* Поддержка SAP-анонсов от VLC
* Добавлена поддержка русских символов в SDP (в кодировке win1251)
* Добавлена поддержка MPEG TS из файлов
